Core Equations

notation

Center Equation law setup

E = 2C

Goldbach becomes a center problem. The even number is the building; C is the address.

Mirror Room room setup

Room(C,g) = (C - g, C + g)

Each valid gap g creates a left wall and right wall around the center.

Mirror Identity proven

(C - g) + (C + g) = 2C = E

The sum never moves. Only the prime/composite state changes.

Left And Right canonical

L_C(g) = C - g
R_C(g) = C + g
L_C(g) + R_C(g) = E

BOTH Indicator success test

BOTH_C(g) = 1_prime(C-g) * 1_prime(C+g)

BOTH_C(g) = 1 iff C-g and C+g are both prime
BOTH_C(g) = 0 otherwise

Rescue Count window count

S(C,W) = sum_{0 <= g <= W} BOTH_C(g)
       = sum_{0 <= g <= W} 1_prime(C-g) * 1_prime(C+g)

S(C,W) >= 1 means rescued inside W

K Window pressure window

W_K(C) = floor(K log^2(C))

The bounded proof target asks whether every large center gets rescued before this window ends.

trueK observed pressure

trueK(C) = firstBothG(C) / log^2(C)

A larger trueK means the first rescue came later relative to scale.

Folded Product structure

(C-g)(C+g) = C^2 - g^2

A BOTH hit is the folded product being prime times prime.

Failure Equation counterexample shape

Goldbach fails at C only if:

for all valid g, BOTH_C(g) = 0

Failure is not random. It means full blocker coverage.

Blocker Flow

what is happening
Valid rooms
Start with parity-valid gaps in the selected window W. These are the rooms that can possibly rescue the center.
Small-prime kill
A small prime p blocks a room when C-g == 0 mod p or C+g == 0 mod p.
Survivors
Rooms not killed by the selected small-prime filter become survivors. Survivor mass is pressure left in the system.
Large-prime lanes
A larger prime p can kill the left side on g = C mod p and the right side on g = -C mod p. Full failure needs these lanes to cover every survivor.
Large-prime close
A survivor can still be composite because of a larger factor. The open problem is proving large blockers cannot close every survivor forever.
Rescue
If any survivor has both sides prime, BOTH_C(g)=1. The center is rescued and Goldbach holds for that E.
Full coverage
If no rescue exists, every valid room is blocked. That exact full-coverage shape is what the proof path must rule out.
